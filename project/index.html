I have developed a project named Travel Guide as part of the Web Development Course (CS5610). The course lectures really helped me to understand the concepts. Then I had applied my learning here in the project.

In this project, a user can do follwing things,

  • Login / Register
  • Start searching for availabe routes between cities
  • Can search for current and future weather information of a city
  • Bookmark his searches
  • Edit his profile information
