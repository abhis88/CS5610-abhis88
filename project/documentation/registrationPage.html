Registration Page

This webpage opens when the user clicks on the right corner Register link. This webpage enables user to register themselves into the system. User need to input their firstname, lastname, email, about me, username and password to register themselves. Below is the screenshot of the registration page.

Details

  • Registration Data is stored in MongoDB
  • The user inputs are saved. Also, it is matched against username to avoid duplicate records.
  • Moves to Profile page after sucessful registration
