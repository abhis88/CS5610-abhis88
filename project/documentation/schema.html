Schema Page

This page tell about the MongoDB schema used in the project. There are in total of two schema used here, UserSchema and CommentSchema.

  • UserSchema

    This schema describes about the user information. Below is how it looks.

    FIELD NAME TYPE
    firstnameString
    lastnameString
    usernameString
    passwordString
    aboutString
    emailString
    favuserArray
    favweatherArray of Objects
    favplacesArray of Objects
  • CommentSchema

    This schema describes about the comment information. Below is how it looks.

    FIELD NAME TYPE
    firstnameString
    commentString
    dateDate
