User Follow Page

You can follow another user. For this you need to logged in and search that user from the Search page (click on the top search icon). Type your user info here. For example I have shown, Abhishek.

Then If you want to see more detail about that user then click on detail button. If you want to like that user then click on Like button. Once, Liked you can unlike that user.

Details

  • You can only like another user if your logged in
  • Once liked you can unlike that user.
  • You can see more details by clicking on detail button
  • All the liked user are saved in your profile
