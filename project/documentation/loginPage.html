Login Page

This webpage opens when the user clicks on the right corner Login link. This webpage enables user to login into the system. User need to input their username and password to login. Below is the screenshot of the login page.

Details

  • Login Data is stored in MongoDB
  • The user credentials are verified against the entries available in the DB
  • Loads all the saved bookmarks and basic profile info after sucessful login
