Project Introduction - Travel Guide

Travel Guide is the project developed as part of Web Development CS5610 Course. I have taken this course in my second semester. This project enables user to search for route information from one city to another. Also, user can see weather deatils of the city of his choice.

List of available services on the website:

SERVICE USE
Route Service You can fetch travel route between any two cities of world.
Weather Service You can fetch weather information of the specified city
User Comment Service You can comment your views on the comment section
User Search Service You can search for other users here.
User follow / User Unfollow Service You can follow or Unfollow other users and see their details.

Various Pages available on website:

WEBPAGE DESCRIPTION
Home Page This is the starting page of my project. You can see in detail about the project on home page.
Login Page This page loads when user clicks on login link. In order to access full version of website you need to be logged in.
Profile Page User personal information is displayed on this page.
Weather Page This page provide you with options to fetch weather information on specified city.
Route Page This page provide you with options to fetch route information between specified cities.
Comment Page This page provide you with options to put your thoughts as a comment.
User Search Page This page provide you with options to fetch details of other users. You can search using firstname, lastname or any bookmarks.
