User Search Page

You can search any user by his firstname, lastname or any bookmarks done by that user. You can even Like or Unlike the searched user if you are logged in. Also you can see more details about the searched user by clicking on details button.

Details

  • Search user based on FirstName.
  • Search user based on LastName.
  • Search user based on any available bookmarks.
  • Like or Unlike user, if your logged in
  • You can see more details by clicking on Detail button
