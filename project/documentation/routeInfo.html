Route Page

Route page gives you an option to fetch available routes between two cities. You need to mention the name of the cities and it will fetch you the details. It also provides you with the google map. Also, if you are logged in you can save your search. Saved serach will be displayed instally under bookmark tab. Moreover, I really appreciate RomeToRio for providing necessary API.

Details

  • This page enables you to fetch route information between two cities.
  • Provides you with google map routes.
  • You can save your search if you are logged in.
