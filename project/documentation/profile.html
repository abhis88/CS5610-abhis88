Profile Page

Profile page contains all the basic information about the logged in user. If user wants then he/she can edit his information by clicking on edit button. Once the editing is complete you can click on done button to save your changes. All your save bookmarks, user can be found on this page.

Details

  • All saved users list can be found here. You can Like or UnLike the user
  • Links to services page are available here.
