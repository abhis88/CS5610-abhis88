API Detail Page

During this project, I have used two API. RomeToRio is an API which provides details about the available route between two cities around the world. I have used this API in the Route page of the project. Secondly, I have use WorldWeatherOnline API which provides the detail of the weather information of any city around the world.

API Details

  • RomeToRio

    Provides details route available between two cities.

  • WorldWeatherOnline

    Provides weather information of any city around the world.
