WEEK 5 : Angular JS and Routing Experiments

In this week, we get to know advanced features of Angular JS like Advanced Angular JS and Routing concepts. After that, I tried to develop some websites.

This is the first website that I developed this week. In this experiment I have used static data stored in Array which acts as a source for the controller. In the second experiment I have used dynamic data using myapifilms api whichs connects to IMDB database and returns the results in JSONP format. In the third experiment I tried to develop a simple Blog Application. A user can write his blog limited to 100 characters and save it on the page although the saved content will not be persisted in database.

In Fourth experiment I have used Angular JS Routing. Routing allows webpage to avoid unnecessary page loads and can dynamically show the results on webpage. In fifth experiment I developed another routing experiment which dyanmically loads pages based on the login credentials. If logged using student credentials, It will show the pages and contents related to students. Similar behaviour can be observed for Facutly and admin credentials.
