Introduction to Project

Project Topic

I will be developing a website that will facilitate the future travellers around the world. Project Name - Traveller's Guide

The main idea is to have following things on my website.

  • Flight tracking system.
  • Current traffic tracking system.
  • Cheap Hotel information
  • City guide system
  • Navigation Map system.
  • Nearest Taxi station.

Technologies to be used:

  • HTML5
  • CSS
  • JQuery / JavaScript
  • NodeJs
  • AngularJS
  • MongoDB
