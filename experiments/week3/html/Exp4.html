Aim of the Experiment
HOME
The main objective of this experiment is to learn basics of jQuery. JQuery plays a vital role in the the construction of a webpage. Using JQuery, we can do various calculations, create different alerts, animations, effects etc on a webpage. JQuery is a very vast area. Below, I have tried to show some features provided by JQuery.
Mouse Location

This shows the location of the mouse.

Auto Complete
This feature enables you to select from the list.
Date Picker
This feature enables you to select date from the calendar.

$("body").mousemove(function (e) {

                     " Y Axis : " + e.pageY);

                        "ActionScript",

                        "PHP",
                        "Python",
                        "Ruby",

                        "Scheme"
];

<!-- To have autocomplete effect -->

    source: availableTags
});

<!-- To enable tooltip -->
$("#tool").tooltip();
})
});
