Aim of the Experiment
HOME
The main objective of this experiment is to learn basics of jQuery. JQuery plays a vital role in the the construction of a webpage. Using JQuery, we can do various calculations, create different alerts, animations, effects etc on a webpage. JQuery is a very vast area. Usually, we need to write different functions for each functionality. Below, I have tried to show draggable features provided by JQuery.
