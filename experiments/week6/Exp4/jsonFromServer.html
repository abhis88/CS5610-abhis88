JSON response using Node JS and Open Shift

Aim of the Experiment

The aim of the experiment is to provide the introduction to Node JS. In this experiment, If you click on the link below then you will be directed to Open Shift Server the data will shown in JSON format. JSON (JavaScript Object Notation) is the open standard format uses human readable text to transmit data object consisting of attribute value pair. It will return ordered list of values. The server checks for a particular url and based on that it sends a response in JSON format to the client.
