Aim of the Experiment

The aim of the experiment is to have a Sign In form in the project homepage. Upon clicking the Sign In button a Sign In modal should appear. This Modal should ask for Username, and Password information. Basic login checks should be implemented.

Here is the Project HomePage. Request you to please click on Sign In button and check the result yourself.

Code Snippet

I have written a simple bootstrap code to have a Modal Sign In functionality that enables the user to login into the page. For more details you can check source code. Here I have shown only the UserName tag of Sign In tab. 
    <div class="modal fade bs-modal-sm" id="myModal" tabindex="-1" role="dialog" aria-labelledby="mySmallModalLabel" aria-hidden="true">
     <div class="modal-dialog modal-sm">
      <div class="modal-content">
       <div class="bs-example bs-example-tabs">
        <ul id="myTab" class="nav nav-tabs">
         <li class="active"><a href="#signin" data-toggle="tab">Sign In</a></li>
        </ul>
      </div>
     <div class="modal-body">
    <div id="myTabContent" class="tab-content">
   <div class="tab-pane fade active in" id="signin">
   
  <form class="form-horizontal">
   <fieldset>
    <div class="control-group">
     <label class="control-label" for="userid">User Name:</label>
      <div class="controls">
       <input required="" id="userid" name="userid" type="text" class="form-control input-medium" placeholder="Abhishek">
      </div>
     </div>
    </fieldset>
   </form>
