Introduction to Project

Project Topic

I will be developing a website that will facilitate the future travellers around the world. Project Name - Travel Guide

The main idea is to have following things on my website.

  • Route Information System
  • Weather Information Syste
  • User Follow / Un Follow other user
  • User comments
  • Search based on User info, bookmark info
  • Login / Registration Option
  • Bookmark / Save user option

Technologies to be used:

  • HTML5
  • CSS
  • JQuery / JavaScript
  • NodeJs
  • AngularJS
  • MongoDB
  • Express JS
