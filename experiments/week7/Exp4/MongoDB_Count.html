MongoDB Count Operation

Aim of the Experiment

MongoDB is a cross-platform document-oriented database. Classified as a NoSQL database, MongoDB eschews the traditional table-based relational database structure in favor of JSON-like documents with dynamic schemas (MongoDB calls the format BSON), making the integration of data in certain types of applications easier and faster.The aim of the experiment is to explain how to count() data using mongodb. Here I am using the same student database that I have used in find experiment. So, lets learn how to count data in mongodb.

Below is the code snippet of the Mongoose which runs on the Open Shift server. This experiment shows the count of the rows from the server. 

server.js code

app.get('/api/Week7/Exp4', function (req, res) {
    studentModel.count(function (err, data) {
        res.json("Count of the rows is "+ data);
    });
});
