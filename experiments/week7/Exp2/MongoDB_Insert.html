MongoDB Insert Operation

Aim of the Experiment

The aim of the experiment is to explain how to insert data using mongodb. Output of this mongodb insert will be a json. So, lets learn how to insert data into mongodb. Below experiment is done both locally and on server.

MongoDB Insert

Click on the link to go to Open Shift server to get response from server. click here. This is how it looks in remote server.

MongoDB Insert (local machine)

This is how it looks locally. You need to start the server mongod.exe. After starting the server you should start the database mongo.exe. Remember to add them in path so that you can run the server and database from any file location.
