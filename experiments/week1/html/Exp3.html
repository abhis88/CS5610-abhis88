There are three ways to implement Cascading Style Sheets (CSS) in HTML pages.
  • In-Line Style
  • CSS in head tag
  • Separate CSS file
Out of all three, Separate CSS file is considered as best practice. Its even better to have a separate folder for CSS as a html file can have more than one css.

Aim of the Experiment

The main objective of this experiment was to learn all three aspects of cascading style sheets. Out of all three, Separate CSS file method is universaly recoginzed for its usefullness. Thus after learning the first and second ways to implement CSS, I have shown the third way in this HTML file. I have implemented all of my Experiments using a separate CSS file.
