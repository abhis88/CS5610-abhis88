Aim of the Experiment

There are various tricks that can be implemented using Cascading Style Sheets (CSS) in HTML pages. Some of those are shown below.

Rotated Block

I am normal div
I am rotated div
Code Snippet
.rotated_div1 { transform: rotate(45deg); }

Zooming on Mouse Hover

Code Snippet
#image1:hover { /*Chrome, IE*/ -webkit-transform: scale(1.25,1.25); -webkit-transition-timing-function:; -webkit-transition-duration: 300ms; /*Opera*/ -o-transform: scale(1.25,1.25); -o-transition-timing-function: ease-in-out; -o-transition-duration: 300ms; /*Mozilla*/ -moz-transform: scale(1.25,1.25); -moz-transition-timing-function: ease-in-out; -moz-transition-duration: 300ms; transform: scale(1.25); z-index: 150; }

360 degree rotation on Mouse Hover

Code Snippet
#image2 { -webkit-transition-duration: 0.9s; -moz-transition-duration: 0.9s; -o-transition-duration: 0.9s; transition-duration: 0.9s; transition-property: transform; }
#image2:hover { -webkit-transform: rotate(360deg); -moz-transform: rotate(360deg); -o-transform: rotate(360deg); }

Opacity Effect on Mouse Hover

Code Snippet
#image3:hover { opacity: 0.5; }
