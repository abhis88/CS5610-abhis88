1. Input Field

Input field allows us to enter some text/value into the field. Usually we find these boxes while filling out a form.

If you use placeholder property of the input tag then you can specify a demo text inside the box so that the user can exactly know what has to be entered.
Also, you can specify the type to be email so that basic validations can be done.
