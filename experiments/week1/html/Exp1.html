Video Tag
Few years back, HTML 5 introduces new tags which is very useful in designing nice webpages. Also, with these new tags, the need for plugins to emmbed videos, audio etc have been eliminated. Below are some of the examples;
Code Snippet
<video width="400" controls>
<source src="../video/video.mp4" type="video/mp4"> Oops ..! Go get a new browser.
</video>
Audio Tag
Similarly, there are various websites that uses different available plugins to play audio on their websites. Thus HTML 5 introduces a seprate and powerful audio tag to eliminate the need of audio plugins. Below are some of the examples;
Code Snippet
<video width="400" controls>
<source src="../video/video.mp4" type="video/mp4"> Oops ..! Go get a new browser.
</video>
iFrame Tag
iFrames can also be used to play youtube or other videos. I was also able to set the width and height of the iFrames. iFrames are recommended over video tags and are widely used these days. Below are some of the examples;
Code Snippet
<iframe width="420" height="315" src="http://www.youtube.com/embed/XGSy3_Czz8k"> </iframe>
