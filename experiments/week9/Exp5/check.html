Project : Autocomplete with Condition

Aim of the Experiment

The aim of the experiment is to see how Autocomplete is implemented. Autocomplete's are used in order to represent multiple information at one place in the form of the list. Here, Source and Destination cannot be same and hence a check has been implemented. Try to select same cities in both the boxes.

Autocomplete

This is used on almost every website on the web. It is very common but very useful. You start typing and you will be provided by various opitions based on your input text as of now. Check below.
