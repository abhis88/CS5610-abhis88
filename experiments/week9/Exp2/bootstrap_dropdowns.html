Project : Bootstrap Dropdowns Experiment

Aim of the Experiment

The aim of the experiment is to see how Bootstrap dropdows are implemented. Dropdowns are used in order to represent multiple information at one place in the form of the list. See below examples for more detail.

Bootstrap Dropdowns

The .dropdown class is used to indicate a dropdown menu. Use the .dropdown-menu class to actually build the dropdown menu.

Normal Dropdown Menu

Use the .dropdown-menu class to actually build the dropdown menu.

  • Web Development
  • Program Design Paradigm
  • Database Management
  • Algorithm

Headers inside the Dropdowns

The .dropdown-header class is used to add headers inside the dropdown menu:

Code Snippet

Below is the code snippet one of the dropdowns. Almost all the dropdowns are written in same manner. Only class names differs. You can get more details on Source Code. 

  <div class="dropdown">
    <h4> Normal Dropdown Menu </h4>
    ...
    <button class="btn btn-default dropdown-toggle" type="button" id="dropdownMenu1" 
                   data-toggle="dropdown" aria-expanded="true">
    Courses
    <span class="caret"> </span>
    </button>
    <ul class="dropdown-menu" role="menu" aria-labelledby="dropdownMenu1">
     <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Web Development</a></li>
     <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Program Design Paradigm</a></li>
     <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Database Management</a></li>
     <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Algorithm</a></li>
    </ul>
