Project : Autocomplete Experiment

Aim of the Experiment

The aim of the experiment is to see how Autocomplete is implemented. Autocomplete's are used in order to represent multiple information at one place in the form of the list. This is used almost on every website. Prime example is google. See below examples for more detail.

Autocomplete

This is used on almost every website on the web. It is very common but very useful. You start typing and you will be provided by various opitions based on your input text as of now. Check below.
