Angular JS MVC Experiment

Aim of the Experiment

The aim of the experiment was to demonstrate the Angular JS MVC functionality. MVC stands for Model-View-Controller. Angular JS extends HTML with new attributes. Angular JS is perfect for Single Page Applications(SPA's). It is a client side JavaScript framework. Thus in the below example, I would like to show the binding property where changing an attribute will be reflected in real-time on the webpage.
  
  
  
  
  
{{employee.firstName + " " + employee.lastName}}
  {{employee.county}}
  {{employee.mobile}}
  {{employee.address}}

Code Snippet

I have implemented the MVC architecture. Here we have a Model and a Controller. Controller acts as a database for the Model. Scope is the scope of the controller. 
var app = angular.module("firstController", [])
app.controller("Controller",
    function ($scope) {
        var employee = {
            firstName: "",
            lastName: "",
            mobile: "",
            county: "",
            address:"",
        };
    $scope.employee = employee;
});
